## Installation

Nippur is available through [CocoaPods](http://cocoapods.org/pods/Nippur). To install
it, simply add the following line to your Podfile:

```ruby
pod "Nippur"
```

Now run on terminal:

```ruby
pod install
```

## Requirements

Nippur is compatible with iOS 6 or later.
